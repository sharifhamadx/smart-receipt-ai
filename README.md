# 🏥 Smart Health Receipts (MediTrack AI)

An AI-powered, privacy-first medical expense tracker that auto-scans receipts, categorizes spending, and converts multi-currency bills in real-time.

## 🚀 Live Demo
**[Click here to try the app!](https://sharifhorizon.surge.sh)**

## ✨ Key Features
- **100% Offline AI Scanning:** Uses Tesseract.js to read medical bills directly in the browser (Maximum Privacy, Zero Cloud Server Costs).
- **Smart Categorization:** Automatically classifies expenses into (Hospital, Pharmacy, Lab, Clinic).
- **Live Currency Conversion:** Detects foreign currencies (USD, EUR, SAR, EGP) and converts them to the local currency (LYD) using real-time rates.
- **Dynamic Localization:** Fully supports Arabic, English, and French dynamically without refreshing.
- **Duplicate Prevention:** Smart hashing prevents logging the same receipt twice.

## 🛠️ Built With
- **React.js** (Frontend & State Management)
- **Tesseract.js** (Local OCR AI)
- **Open ER-API** (Live Exchange Rates)
- **Engineered entirely on a 6-inch smartphone using Termux!**

## 📖 The Story
This project was born out of resilience. Built by a refugee navigating infrastructure challenges (frequent power outages, unstable internet), this app proves that constraint breeds innovation. Moving away from heavy cloud LLMs, the architecture was pivoted to a local-first AI engine, entirely coded, debugged, and deployed from a mobile device.
