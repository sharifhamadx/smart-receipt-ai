# 🏥 Smart Health Receipts (MediTrack AI)

An AI-powered, privacy-first medical expense tracker that auto-scans receipts, categorizes spending, and converts multi-currency bills in real-time. This project features a robust **Advanced Trash System with a 12-month retention policy** and leverages **native IndexedDB** to handle vast amounts of data efficiently on the client-side.

## 🚀 Live Demo
**[Click here to try the app!](https://sharifhorizon.surge.sh)**

## ✨ Key Features
- **100% Offline AI Scanning:** Uses Tesseract.js to read medical bills directly in the browser.
- **Advanced Trash System:** Soft delete logic with 12-month auto-cleanup.
- **Smart Categorization:** Automatically classifies expenses.
- **Live Currency Conversion:** Real-time exchange rates.
- **Dynamic Localization:** Fully supports Arabic, English, and French.

## 🛠️ Built With
- **React.js & IndexedDB**
- **Tesseract.js**
- **Engineered entirely on a smartphone using Termux!**
