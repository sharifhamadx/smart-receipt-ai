# 🏥 Smart Health Receipts

An AI-powered, privacy-first medical expense tracker that auto-scans receipts, categorizes spending, and converts multi-currency bills in real-time.

## 🚀 Live Demo
**[Click here to try the app!](https://sharifhorizon.surge.sh)**

## ✨ Key Features
- **100% Offline AI Scanning:** Uses Tesseract.js to read medical bills directly in the browser.
- **Complete Data Control:** View all historical scans with an option to permanently delete records from the local database.
- **Smart Categorization:** Automatically classifies expenses into categories.
- **Live Currency Conversion:** Detects foreign currencies and converts them instantly.
- **Dynamic Localization:** Fully supports Arabic, English, and French.

## 🛠️ Built With
- **React.js & LocalStorage**
- **Tesseract.js**
- **Engineered entirely on a smartphone using Termux!**
